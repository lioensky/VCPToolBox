【美学本质】
一句话：把「Tip 3: This course is about relationships. All colors used in a single presentation must be related. In the example above, orange is related」读成一个结构问题：你要做的不是复述步骤，而是让关系在边界处变得可见。

【原文触发】

## 品味裁决

【俗套陷阱】
只抄表面形态：让内容变成‘像’，却无法解释‘为什么像’。

【优雅签名】
把现象还原成关系：谁改写谁、改写多少、何时失效——写清楚边界。

---

## 张力透视

【张力结构】
血缘（成分） vs 领养（拼贴）

【统一机制】
用 parent/child 的成分与区间约束，把‘看起来像’变成可推断的关系。

【平衡临界点】
child 若不含 parents 成分或 value 不在区间内，现象会突然失效。

---

## 跨媒介迁移

【灵魂（不变量）】
高级不是更强烈，而是更能被结构证明。

【皮相变体】
- 品牌：主辅色要‘有血缘’。
- 插画：阴影色需含局部色亲缘。
- UI：中间态在不同背景下偏向端点。

【边界条件】
当 block 的信息只剩图像/口号时，必须回到‘对照 + 变量控制’重建证据。

## 审美唤醒

【认知改变】
- 之前：我记住了一个说法。
- 之后：我掌握了一个可复用模型。

【审美模型】
- 模型名称：亲缘-区间闸门
- 核心公式：错觉发生 = 亲缘成立 ∧ value 区间成立 ∧ 比较路径成立
- 适用范围：绘画、摄影、UI 的现象重建与评审。
- 限制条件：依赖清晰边界与一致性；脱离语境会失效。

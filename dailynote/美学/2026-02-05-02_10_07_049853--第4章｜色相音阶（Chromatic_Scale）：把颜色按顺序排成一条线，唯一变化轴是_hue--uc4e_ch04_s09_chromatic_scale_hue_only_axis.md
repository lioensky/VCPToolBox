# 第4章｜色相音阶（Chromatic Scale）：把颜色按顺序排成一条线，唯一变化轴是 hue

## 美学本质

一句话：Chromatic scale 就是“按光谱顺序排列的一串 hue”——它允许复杂与稀释，但它的身份条件只有一个：序列推进必须体现在 hue 的变化上。

原文给出的定义非常干净：
- 任何按 spectrum order 排列的线性 hue 序列，都叫 chromatic scale；
- 它可以展示纯饱和色，也可以展示更复杂的颜色；
- 但它的“判据”是：这条序列的进程体现为 hue 的连续变化。

## 品味裁决

【俗套陷阱】把渐变当装饰：只追求“顺滑”，不管 hue 的顺序与间隔。
为何廉价：你做的是“模糊”，不是“音阶”——观者感不到方向，只感到糊。

【优雅签名】把 chromatic scale 当叙事：让 hue 的顺序决定情绪迁移（从冷到暖、从近似到对立）。
精妙之处：你获得一种可控的“色彩行进”，像旋律一样可预期、可回忆。

## 张力透视

【张力结构】线性推进（方向感） vs 色轮回环（循环感）。
【统一机制】线用于“旅程”，轮用于“关系”：同一套 hue，放在不同结构里，叙事语义不同。
【平衡临界点】推进太快会突兀；推进太慢会无聊——关键是让间隔可被感知。

## 跨媒介迁移

【灵魂（不变量）】有序变化会被大脑读成“时间/节奏”。
【皮相变体】
- 电影/摄影：色相从蓝到橙常被读作从疏离到亲密；
- UI：状态过渡用 hue 的方向来降低学习成本；
- 音乐：音阶是频率的有序变化，chromatic scale 是 hue 的有序变化。
【边界条件】当同时大幅改变明度/饱和度时，hue 的变化会被掩盖；要保证“主轴”清晰。

## 审美唤醒

【审美模型】方向感 ≈ hue 间隔可辨识度 × 顺序一致性。
【一句话启发】「让颜色像旋律：一切变化都围绕 hue 的顺序发生。」

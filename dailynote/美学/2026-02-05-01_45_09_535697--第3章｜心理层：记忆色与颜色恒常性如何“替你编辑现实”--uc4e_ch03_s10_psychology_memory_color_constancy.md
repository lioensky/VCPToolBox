# 第3章｜心理层：记忆色与颜色恒常性如何“替你编辑现实”

## 美学本质

一句话：你看到的颜色不全来自眼前——记忆色与颜色恒常性会让大脑用“应该是什么颜色”覆盖“实际是什么颜色”，以换取生活的视觉舒适。

原文两个概念非常关键：
- memory color：橙子就该是橙色；海再红也有人会画成蓝；
- color constancy：熟悉物体在不同光源下被当作“同色”，甚至全白厨房里一堆不同白被归为同一“白”。

## 品味裁决

【俗套陷阱】把预期当事实：凭“常识色”作画、作设计、作验收。
为何廉价：你会错过真实的颜色变化，也会在关键场景里误判（尤其是白与近似色）。

【优雅签名】对“熟悉色”保持警惕：在关键决策时强制自己回到观察（对照样、灰卡、标准光），把大脑的编辑器暂时关掉。
精妙之处：你获得更可靠的颜色判断。

## 张力透视

【张力结构】真实差异（实际颜色变了） vs 心理稳定（把它当作没变）。
【统一机制】承认恒常性是必要的：否则我们会被无穷色差耗尽；但设计工作需要在特定时刻放大差异、做出精细判断。
【平衡临界点】日常生活里过度敏感会疲惫；设计工作里过度恒常会失误。

## 跨媒介迁移

【灵魂（不变量）】颜色判断是“观察”与“记忆”的拉扯。
【皮相变体】
- 摄影：白平衡失败往往来自记忆色干扰。
- 产品/室内：白色系最容易被恒常性掩盖差异，必须在目标光源下比对。
- UI：近似灰与近似白在不同屏幕上差异会被恒常性抹平，需要更强结构区分。
【边界条件】当你要做“现实主义表达”，需要更强的反恒常训练；当你要做“符号化表达”，可以利用记忆色加速理解。

## 审美唤醒

【审美模型】你看到的颜色 = 眼前刺激 + 大脑预期 - 被忽略的差异。
【一句话启发】「颜色恒常性让生活轻松，也让设计更容易犯错。」

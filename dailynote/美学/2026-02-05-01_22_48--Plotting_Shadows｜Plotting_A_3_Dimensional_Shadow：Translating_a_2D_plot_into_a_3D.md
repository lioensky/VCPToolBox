# Plotting Shadows｜Plotting A 3 Dimensional Shadow：Translating a 2D plot into a 3D plot requires an additional plot.

## 美学溯源

【美学本质】
一句话：把「Plotting A 3 Dimensional Shadow」读成一个结构问题：你要做的不是复述步骤，而是让关系在边界处变得可见。

【原文触发】
- Translating a 2D plot into a 3D plot requires an additional plot. If a form exists on a 3 dimensional plane, unlike the 2D one above, both the form and the light source must be set in a 3D space. Visualizing it in a box might make the concept clearer.
- Step 2. Connect both (Bs) with a straight line and carry this line out to the right until it meets line (LT). Point X is the spot where these two lines meet. The line from the (B) or bottom of the pole to X is the length and direction of the shadow.
- Note: In the illustration on the left, the shadow of a wall is plotted. If each end is viewed as a post, with a Top (T) and Bottom (B), the two X points are found. Simply join the X points and the shape of the cast shadow is revealed.

## 品味裁决

【俗套陷阱】
只抄表面形态：让内容变成‘像’，却无法解释‘为什么像’。

【优雅签名】
把现象还原成关系：谁改写谁、改写多少、何时失效——写清楚边界。

## 张力透视

【张力结构】
命名冲动（直接画物） vs 关系观察（先画现象）

【统一机制】
透明不是材质贴图，而是覆盖规则：被覆盖的颜色必须被同样方式改写。

【平衡临界点】
只要一处不一致，整张膜都会被判为假。

## 跨媒介迁移

【灵魂（不变量）】
真实感来自一致性：规则统一统治所有区域。

【皮相变体】
- UI：玻璃拟态看一致改写。
- 摄影：滤镜必须同改全场。
- 绘画：薄纱/雾气=覆盖规则。

【边界条件】
当 block 的信息只剩图像/口号时，必须回到‘对照 + 变量控制’重建证据。

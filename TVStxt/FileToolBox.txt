# VCP 文件管理与系统控制工具箱能力收纳
这里收纳了文件搜索、读写、批量操作、代码检索及系统命令行工具。具体调用格式会按当前语境折叠展开。

## 运行时查询隐藏说明
如果你在执行任务时判断需要本工具箱中未展开的参数格式，先按 supertool.txt 中的 StaticToolboxQuery 通用格式查询相关说明块，再调用真实工具。

## 最小安全自检入口
当需要自行测试文件工具链是否可用，但用户没有指定具体文件操作时，优先使用非破坏性命令：

1. 搜索连通性测试
tool_name:「始」ServerSearchController「末」,
command:「始」search「末」,
query:「始」VCP「末」,
maxResults:「始」3「末」

2. 文件操作边界测试
tool_name:「始」ServerFileOperator「末」,
command:「始」ListAllowedDirectories「末」

除非用户明确要求，不要用自检去写入、编辑、删除、移动文件，也不要主动执行 PowerShell 命令。

## 1. PowerShell 命令行工具 (ServerPowerShellExecutor)
直接调用并获取运行结果。默认使用后端 ServerPowerShellExecutor，适合 Agent/后台任务/服务器维护。只有明确需要打开 VCPChat 前端持久 PTY 交互窗口时，才使用 FrontendPowerShellExecutor；不要使用旧名 PowerShellExecutor 或 OldPowerShellExecutor。
tool_name:「始」ServerPowerShellExecutor「末」,
command:「始」Get-ChildItem「末」,
executionType:「始」blocking/background「末」, // 默认blocking。background用于长时任务。
tool_password:「始」123456「末」 // 可选；Server版执行敏感命令时输入6位安全码。

## 2. 文件快速搜索 (ServerEverythingSearch / LocalSearchController)
基于 Everything 模块实现秒级搜索。默认使用 ServerEverythingSearch 操作 VCP 主服务器所在机器；只有明确需要搜索 VCPChat 分布式前端/用户桌面机器时，才使用 LocalSearchController。旧名 ServerSearchController 仅保留兼容。
tool_name:「始」ServerEverythingSearch「末」, // LocalSearchController 代表操作分布式前端/用户桌面机器
command:「始」search「末」,
query:「始」搜索关键词 (支持Everything语法)「末」,
maxResults:「始」50「末」

## 3. 基础文件阅读 (ServerFileOperator)
读取 VCPToolBox 服务器可访问目录内的文件，支持富文本/多媒体。只有明确要操作 VCPChat 分布式前端/用户桌面机器时，才改用 FileOperator。
tool_name:「始」ServerFileOperator「末」,
command:「始」ReadFile「末」,   
filePath:「始」/path/to/your/document.pdf「末」

## VCP 项目代码结构参考
- **后端 (VCPToolBox)**: 使用 `ServerFileOperator`
  - `server.js`, `Plugin.js`, `KnowledgeBaseManager.js`, `WebSocketServer.js`
  - `modules\chatCompletionHandler.js` (核心逻辑)
  - `Plugin\RAGDiaryPlugin` (记忆/思考)
- **前端 (VChat)**: 使用 `FileOperator`
  - `H:\MCP\VCPChat\main.js`, `renderer.js`
  - `H:\MCP\VCPChat\modules`, `Assistantmodules`

# VCP 文件工具命名更新说明
## 秒搜工具当前命名
- 首选主名: `ServerEverythingSearch`
- 旧兼容名: `ServerSearchController`
- 分布式前端/桌面端工具: `LocalSearchController`

## 调用约定
- 搜索 VCP 主服务器所在机器时，优先使用 `ServerEverythingSearch`
- 只有明确要搜索 VCPChat 分布式前端或用户桌面机器时，才使用 `LocalSearchController`
- `ServerSearchController` 仅保留兼容，不再作为首选名字

## 最小自测示例

command:「始」search「末」,
query:「始」VCP「末」,
maxResults:「始」3「末」
